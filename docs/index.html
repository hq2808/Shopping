Dành cho SA / Performance QA

1. Kiểm thử Kiến trúc & Độ Bền Bỉ (SA & Performance Test Guide)

Tài liệu chuyên sâu cung cấp đặc tả sơ đồ liên kết mạng, cấu hình retry RabbitMQ, thuật toán Redis Lua, cấu hình HikariCP, và các kịch bản kiểm thử tranh chấp (Race Conditions), thảm họa hạ tầng (Chaos Engineering):

  • Kiểm thử tranh chấp mua hàng đồng bộ qua DB (Pessimistic Lock) và bất đồng bộ (Redis RAM).
  • Xác thực cơ chế xử lý trùng lặp tin nhắn hàng đợi (Idempotency) và độ trễ hàng đợi (Lag).
  • Mô phỏng thảm họa hạ tầng (sập DB PostgreSQL, crash cache Redis RAM) và tự phục hồi.
  • Sử dụng wrk/JMeter tạo tải cực hạn và chạy câu lệnh SQL giám sát nghẽn database lock.
  • Lộ trình phát triển chuyên sâu từ Manual/API Tester lên System Architect QA.

Dành cho Backend Developer

5. Đánh giá Khả năng Chịu tải DB (Scalability Database Audit)

Báo cáo đánh giá chi tiết về các lỗi nghẽn cổ chai vật lý tại cơ sở dữ liệu PostgreSQL khi đối mặt với hàng triệu sản phẩm và người dùng đồng thời:

  • Nguy cơ tranh chấp ghi (Race Condition) và bán lố tồn kho (Over-selling).
  • Hiện tượng quá tải CPU do quét toàn bộ bảng (Full Table Scan) khi tìm kiếm sản phẩm.
  • Cải tiến an toàn và tốc độ sử dụng Khóa Bi quan (Pessimistic Write Lock) trong Spring JPA.
  • Cấu hình tối ưu hóa Connection Pool HikariCP chuyên nghiệp cho dự án triệu tải.

6. Giải pháp Kiến trúc Giảm chấn Chịu tải cực hạn (High-Concurrency)

Tài liệu giải pháp phân tích chuyên sâu về kiến trúc chịu tải triệu yêu cầu giao dịch/giây, đưa cơ sở dữ liệu PostgreSQL về vùng an toàn tuyệt đối:

  • Nguyên lý khấu trừ tồn kho nguyên tử tốc độ cao trong bộ nhớ đệm RAM bằng Redis và Lua Script.
  • Kiến trúc ghi bất đồng bộ (Write-Behind) đẩy đơn hàng vào Message Queue (Kafka/RabbitMQ).
  • Sơ đồ và phân tích so sánh chi tiết hiệu năng giữa các giải pháp khóa truyền thống và kiến trúc đệm MQ.

Dành cho System Architect

7. Phân tích Kiến trúc Pipeline Tìm kiếm (Search Indexing Pipeline)

Tài liệu phân tích chuyên sâu về kiến trúc đồng bộ dữ liệu thời gian thực từ MongoDB sang Elasticsearch của cấu phần main-feature-search:

  • Chiến lược Initial Load (Backfile) tối ưu hóa cài đặt index và keyset pagination.
  • Chiến lược Change Stream CDC thời gian thực, buffering và checkpointing trên Redis.
  • Cơ chế tự động khôi phục dữ liệu chống mất mát khi Oplog của MongoDB bị rollover.
  • Quy trình tiền xử lý, trích xuất tag giới tính và index hashtags tự động bằng UUIDv5.
  • Hướng dẫn chi tiết từng bước lập trình và code lại pipeline bằng Java Spring Boot (Spring Reactor, Redis State, ES Java Client).

8. Lộ trình ôn tập phỏng vấn Java Senior (Java Prep Hub)

Tài liệu chuyển đổi dự án Main Server sang Java/Spring Boot để củng cố kiến thức System Design nâng cao phục vụ ôn phỏng vấn:

  • Tái cấu trúc mã nguồn (Refactoring & Clean Code) với Strategy + Autowired Map.
  • HTTP/2/3, cấu hình ShallowEtagHeaderFilter, JWT stateless authentication, HSTS bảo mật.
  • Aggregation Pipelines & Transactions trong MongoDB, tối ưu hóa Hibernate (Join Fetch, Entity Graph).
  • Hàng đợi bất đồng bộ với ThreadPoolTaskExecutor, khóa phân tán ShedLock + Redis cho Scheduled task.
  • Chuyển đổi luồng upload GCS, transcode video DRM, checkout thanh toán So-net và livestream WebSocket.
