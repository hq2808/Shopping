Dành cho QA / Tester

0. Hướng dẫn Khởi chạy Hệ thống (Quick Start Guide)

Tài liệu hướng dẫn chi tiết các bước thiết lập môi trường và kích hoạt hệ thống trong vòng 2 phút bằng Docker Compose:

  • Cài đặt Docker Desktop và chuẩn bị tệp cấu hình chạy nhanh.
  • Sử dụng cấu hình docker-compose để tải tự động image từ xa.
  • Các thông số kết nối Database Client (DBeaver, pgAdmin) để đối soát dữ liệu.
  • Hướng dẫn dọn dẹp và giải phóng tài nguyên sau khi hoàn thành kiểm thử.

Dành cho QA / Tester

1. Hướng dẫn Kịch bản Kiểm thử Tính năng (Postman Guide)

Tài liệu biên soạn chi tiết các kịch bản kiểm thử API thủ công thông qua Postman, hướng dẫn thiết lập môi trường, danh sách dữ liệu mẫu và các bước xác thực:

  • CRUD và kiểm tra validation biên lỗi đầu vào của các sản phẩm.
  • Đăng ký và xác thực an toàn bằng JWT Bearer Token (kiểm tra phân tích claims).
  • Kiểm tra phân quyền nghiêm ngặt theo vai trò hệ thống (ROLE_USER, ROLE_ADMIN).
  • Kịch bản kiểm thử tính toàn vẹn dữ liệu giao dịch đặt hàng (@Transactional Rollback).

Dành cho Backend Developer

2. Đánh giá Khả năng Chịu tải DB (Scalability Database Audit)

Báo cáo đánh giá chi tiết về các lỗi nghẽn cổ chai vật lý tại cơ sở dữ liệu PostgreSQL khi đối mặt với hàng triệu sản phẩm và người dùng đồng thời:

  • Nguy cơ tranh chấp ghi (Race Condition) và bán lố tồn kho (Over-selling).
  • Hiện tượng quá tải CPU do quét toàn bộ bảng (Full Table Scan) khi tìm kiếm sản phẩm.
  • Cải tiến an toàn và tốc độ sử dụng Khóa Bi quan (Pessimistic Write Lock) trong Spring JPA.
  • Cấu hình tối ưu hóa Connection Pool HikariCP chuyên nghiệp cho dự án triệu tải.

Dành cho System Architect

3. Giải pháp Kiến trúc Giảm chấn Chịu tải cực hạn (High-Concurrency)

Tài liệu giải pháp phân tích chuyên sâu về kiến trúc chịu tải triệu yêu cầu giao dịch/giây, đưa cơ sở dữ liệu PostgreSQL về vùng an toàn tuyệt đối:

  • Nguyên lý khấu trừ tồn kho nguyên tử tốc độ cao trong bộ nhớ đệm RAM bằng Redis và Lua Script.
  • Kiến trúc ghi bất đồng bộ (Write-Behind) đẩy đơn hàng vào Message Queue (Kafka/RabbitMQ).
  • Sơ đồ và phân tích so sánh chi tiết hiệu năng giữa các giải pháp khóa truyền thống và kiến trúc đệm MQ.
